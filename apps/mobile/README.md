# ERKONA

ERKONA — платформа для специалистов индустрии красоты и их клиентов.

## Наша цель

Создать удобное приложение, которое объединит:

- 📅 Онлайн-запись
- 💬 Общение мастера и клиента
- 🔔 Push-уведомления
- 👥 Базу клиентов
- 📱 Современный мобильный интерфейс

## Статус проекта

🚧 Активная разработка
